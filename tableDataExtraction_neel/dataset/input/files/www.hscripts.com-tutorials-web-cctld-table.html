Country Code Top Level Domain - ccTLD Table?




Explanation

ccTLD:
In addition to the generic top level domain there are TLDs with two letters (such as .in, .de, and .jp) that have been established for over 240 countries and are referred to as "country-code" Top Level Domain or "ccTLDs".
